# Tinkoff_Sirius_ML_2023_CV

### Для начала нужно описать инопланетян.


1. Инопланетяне питаются энергией солнца.
2. Инопланетяне воспринимают формы лучше чем цвета.
3. Инопланетяне могут проходить сквозь объекты.
